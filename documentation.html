Installations

  1. Before install System Information, make sure you already installed latest vQmod (minimum v2.4.1).
  2. Extract the archive and copy all file/ folder inside upload folder into your site root.
  3. Set the group permission to allow access and modify System Information.
  4. System Information ready to use now, go to System > System Infomation.
 

Updates

  1. To updates, extract the archive and copy all file/ folder inside upload folder into your site root.
 

Usages

  1. System Information is an extensions designed to provide information about your hosting related to OpenCart requirements.
  2. To access System Information, simply go to System > System Infomation
 

Information

  1. For any support question or suggestion, you can contact us through Ticket Support (login required).
  2. For further documentation please refer to System Information Documentation.
