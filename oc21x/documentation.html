Requirements

  1. OpenCart v2+
  2. vQmod v2.5.0+
 

Installations

  1. System Information require vQmod to work properly. Download and install latest vQmod before proceed installation.
  2. Download and extract System Information.

  3. Uploading

    Two System Information uploading method, with Extension Installer and Manually through FTP (prefered).

    1. Use Extension Installer to upload sysinfo.ocmod.zip. (Use 2nd method below if you had issue with this method)
    2. Manually with FTP software. Upload all files INSIDE the upload/ folder to your site root.
  4. Add Access permission at System > User > User Groups
  5. System Information ready to use, go to Sidebar Menu > "SI" (System Information) to access.
 

Updates

Update have same step as Installation
